[SIZE="4"][B][COLOR="darkorange"]BetterUI - Gamepad Interface Overhaul[/COLOR][/B][/SIZE]

BetterUI modernizes ESO's gamepad UI with cleaner navigation, richer tooltips, configurable layouts, and high-impact quality-of-life upgrades for Inventory, Banking, and Resource Frames.

[COLOR="darkorange"][SIZE="3"][B]What's New in v3.00 (Major Update)[/B][/SIZE][/COLOR]
[LIST]
[*]Major system architecture migration offering increased UI performance and error handling
[*]Complete modernization of Resource Orb Frames with new high-res textures, fluid animations, and combat indicators
[*]New high-fidelity scroll indicators and gradient selection highlights for precise, full-row visibility
[*]Unified header-sort controllers with intuitive visual indicators and convenient Y-menu sort actions
[*]Inventory, Craft Bag and Banking multi-select actions with smart quantity dialogs and capacity monitoring
[*]Language-aware font system preserving localized text behavior across multiple languages
[*]Currency refresh: Trade Bars terminology update, Seals of Endeavor, and Tome Points support
[/LIST]

[COLOR="darkorange"][SIZE="3"][B]Requirements[/B][/SIZE][/COLOR]
[LIST=1]
[*][URL="https://esoui.com/downloads/info7-LibAddonMenu.html"]LibAddonMenu-2.0[/URL] is required for BetterUI to load and expose settings.
[/LIST]
